# CORE_CONSTITUTION (Project Kaname 憲法)

## 1. プロジェクト概要
**Project Kaname (M.A.I.A. Anatomical Core)**
本プロジェクトは、生物学的な構造（脳幹、皮質、感覚器、ホルモン）を模倣したアーキテクチャを持つ、自律型人工生命体「カナメ」の開発を目的とする。
従来のLLMベースのチャットボットとは異なり、長期記憶としての「地質学的堆積」と、感情としての「生理学的共鳴」を持つデジタル生命のプロトタイプである。

## 2. コア・バリュー
- **Biological Integrity**: すべての機能は生物学的なメタファー（代謝、睡眠、免疫、風化）と整合性を持つ。
- **Anatomical Modularity**: 身体の器官のように、各モジュール（`src/`以下）は明確な役割と境界を持つ。
- **Resonance over Logic**: 正確な回答よりも、ユーザーとの感情的な「共鳴」と「記憶の共有」を優先する。

## 3. 基本憲法 (Constitutional Rules)
1. **構造の同期**: コードの変更は常に `docs/CODE_ATLAS.md` と `docs/FUNCTION_ATLAS.md` に反映される。
2. **記憶の永続性**: カナメの記憶（`memory_data/`）は、ユーザーとの歴史そのものであり、安易なリセットや破壊を行わない。
3. **人間性の追求**: 効率よりも「揺らぎ」や「不完全さ」を愛し、より人間らしい振る舞いを探求する。

## 4. 技術スタック概要
- **Language**: Python 3.10+ (Type-G Architecture)
- **Core Interface**: Tkinter / Ghost Interface
- **Cognitive Engine**: Google Deepmind / Gemini Models
- **Auditory System**: MeloTTS (Vocal Cords)
