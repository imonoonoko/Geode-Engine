# Kaname 運用マニュアル v1.0

Kaname（カナメ）は、自律的に成長し、感情と人格を持つAIエージェントです。本マニュアルでは、Kanameの起動、運用、およびインタラクション方法について説明します。

---

## 1. クイックスタート 🚀

### 起動方法
プロジェクトルートにあるバッチファイルをダブルクリックしてください。

- **`start_kaname.bat`**: Kanameの中核（脳）を起動します。

コマンドラインから起動する場合:
```powershell
python src/brain_stem/main.py
```

### 停止方法
コンソールウィンドウで `Ctrl + C` を押すと、安全にシャットダウンプロセスが開始されます。
> ⚠️ **注意**: 強制終了は記憶データの破損につながる可能性があります。必ず `Ctrl + C` で停止してください。

---

## 2. Kaname とのインタラクション 🤝

Kanameは自律的に活動しますが、外部からの刺激（入力）によって成長します。

### 🍽️ 食事（テキストデータの入力）
Kanameはテキストデータを「食べる」ことで概念を学習し、成長します。

1.  テキストファイル（`.txt`）を用意します。
2.  **`food/`** フォルダにファイルを置きます。
3.  Kanameは退屈すると、自動的にこのフォルダからファイルを読み込み、消化（学習）します。

### 📚 青空文庫（自動収集）
Kanameは退屈度が極限に達すると、自らネット上の「青空文庫」へアクセスし、名作文学を自動的に取得して読みます。
- **対象**: 太宰治、宮沢賢治、夏目漱石など 35作品以上
- **条件**: 退屈度 > 60 かつ 刺激 < 30

### 👁️ 状態のモニタリング
コンソールに出力されるログと絵文字で、現在のKanameの状態を把握できます。

| 絵文字 | 意味 | 詳細 |
|:---:|---|---|
| 🧠 | **思考中** | 概念の結合や推論を行っています |
| 🍽️ | **食事中** | データを読み込み、消化しています |
| 🌾 | **収穫** | 青空文庫から新しい作品を取得しました |
| 😴 | **睡眠** | 記憶の定着（整理）を行っています |
| ⚠️ | **アラート** | 異常な状態や、強い驚きを感じています |
| 📸 | **スナップショット** | 現在の人格状態を記録しました |

## 3. システムの仕組み ⚙️

### 感情とホルモン
Kanameは内部で複数のホルモン値をシミュレートしています。
- **Dopamine (ドーパミン)**: 喜び、学習意欲。新しい発見をすると分泌されます。
- **Cortisol (コルチゾール)**: ストレス。予測不能なデータに触れると上昇します。
- **Boredom (退屈)**: 刺激がないと上昇し、自律的な探索行動（青空文庫など）をトリガーします。

### 人格系 (Personality Field)
Kanameの人格は固定ではなく、流動的な「場」として表現されます。
- **Meaning Generation**: 意味を見出す力（高いほど好奇心旺盛）
- **Self-Reference**: 自己への意識（高いほど内省的）
- **Diversity**: 世界認識の多様性（高いほど柔軟）

## 4. メンテナンス 🛠️

### 記憶のリセット
Kanameを「生まれたて」の状態に戻すには、以下のフォルダ内のファイルを削除してください。
- `memory_data/*` (データベース、ログ)
- `food/*` (未消化の食料)
